# Python ML Libraries Knowledge Base 📚

Welcome, explorer! This knowledge base is your ultimate companion on your journey to becoming a Machine Learning Engineer. We've gathered 11 essential libraries that will take you from basic math to deploying powerful models in production.

Each folder contains an overview to get you started, a handy cheatsheet for those "how do I do this again?" moments, common patterns used in the real world, and a list of common pitfalls to help you avoid those pesky beginner mistakes.

Happy learning! You've got this! ✨

### 📦 Essential Libraries
- [NumPy](./numpy/overview.md) - The foundation of numerical computing.
- [Pandas](./pandas/overview.md) - The gold standard for data manipulation.
- [Matplotlib](./matplotlib/overview.md) - The classic plotting library for Python.
- [Seaborn](./seaborn/overview.md) - Beautiful statistical visualizations made easy.
- [Scikit-Learn](./scikit-learn/overview.md) - Machine Learning for everyone.
- [PyTorch](./pytorch/overview.md) - Cutting-edge Deep Learning.
- [Jupyter](./jupyter/overview.md) - The interactive playground for data science.
- [MLflow](./mlflow/overview.md) - Tracking and managing your experiments.
- [Evidently](./evidently/overview.md) - Monitoring your models in production.
- [FastAPI](./fastapi/overview.md) - Building high-performance ML APIs.
- [Docker](./docker/overview.md) - Packaging your apps to run anywhere.
