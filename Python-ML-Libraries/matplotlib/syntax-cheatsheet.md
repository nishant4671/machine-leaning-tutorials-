# Matplotlib Syntax Cheatsheet 📝

| Signature | Description | Tiny Example |
| :--- | :--- | :--- |
| `plt.plot(x, y)` | Line plot | `plt.plot([1, 2], [3, 4])` |
| `plt.scatter(x, y)` | Scatter plot | `plt.scatter(x, y)` |
| `plt.bar(x, height)` | Bar chart | `plt.bar(names, vals)` |
| `plt.hist(data)` | Histogram | `plt.hist(values, bins=10)` |
| `plt.title('Text')` | Add a title | `plt.title('Sales')` |
| `plt.xlabel('Text')` | Label for X axis | `plt.xlabel('Time')` |
| `plt.ylabel('Text')` | Label for Y axis | `plt.ylabel('Price')` |
| `plt.legend()` | Show labels for data | `plt.legend()` |
| `plt.show()` | Display the plot | `plt.show()` |
| `plt.savefig('n.png')` | Save plot to file | `plt.savefig('plot.png')` |
| `plt.subplot(r, c, n)` | Create multiple plots | `plt.subplot(1, 2, 1)` |
| `plt.grid()` | Show grid lines | `plt.grid(True)` |
| `plt.xlim(min, max)` | Set X axis limits | `plt.xlim(0, 100)` |
| `plt.clf()` | Clear the current figure | `plt.clf()` |
