# Scikit-Learn Overview 🤖

Welcome to the heart of Machine Learning in Python! 

### What is Scikit-Learn?
Scikit-Learn (sklearn) is a robust library for classical machine learning. It covers everything from data preprocessing to model evaluation.

### Why use it?
It has a incredibly consistent "API". Once you learn how to use one model (like Linear Regression), you practically know how to use them all (Random Forests, SVMs, etc.).

### Core Concepts
1. **Estimators**: Objects that learn from data (`.fit()`).
2. **Transformers**: Objects that modify data (`.transform()`).
3. **Predictors**: Objects that make guesses (`.predict()`).

### Installation
```bash
pip install scikit-learn
```

### Alternatives
- **XGBoost/LightGBM**: For more powerful gradient boosted trees.
- **PyTorch/TensorFlow**: For Deep Learning and Neural Networks.
