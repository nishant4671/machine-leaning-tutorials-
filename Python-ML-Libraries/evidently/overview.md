# Evidently Overview 🔍

Is your model still working as well as it did yesterday? Evidently helps you find out.

### What is Evidently?
Evidently is an open-source library for monitoring machine learning models in production. It generates interactive reports on data drift and model performance.

### Why use it?
Models "drift" over time—the real world changes, and your model might become obsolete. Evidently catches this before it becomes a problem for your business.

### Core Concepts
1. **Reports**: High-level summaries of data and model health.
2. **Test Suites**: Automated "Pass/Fail" checks for your data.
3. **Data Drift**: Detecting when the input data distribution changes significantly.

### Installation
```bash
pip install evidently
```

### Alternatives
- **Deepchecks**: Great for data validation in research.
- **Whylogs**: For massive scale data logging.
