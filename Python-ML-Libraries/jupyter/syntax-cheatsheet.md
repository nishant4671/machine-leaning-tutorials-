# Jupyter Syntax & Shortcuts 📝

| Command/Key | Description |
| :--- | :--- |
| `Shift + Enter` | Run cell and move to next |
| `Ctrl + Enter` | Run cell and stay |
| `Alt + Enter` | Run cell and insert new one below |
| `Esc` | Enter Command Mode |
| `Enter` | Enter Edit Mode |
| `A` (in cmd mode) | Insert cell Above |
| `B` (in cmd mode) | Insert cell Below |
| `D, D` (in cmd mode)| Delete selected cell |
| `M` (in cmd mode) | Change cell to Markdown |
| `Y` (in cmd mode) | Change cell to Code |
| `%timeit` | Measure execution time of a line |
| `%%timeit` | Measure execution time of a whole cell |
| `?function_name` | Show documentation for a function |
| `!ls` or `!dir` | Run a shell command |
