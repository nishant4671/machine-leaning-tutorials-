# Jupyter Pro Tips 🚀

Here are 5 must-know magics and shortcuts for any ML Engineer:

1. **%matplotlib inline**: Ensures your plots appear directly below your code cells.
2. **%timeit**: Put this before a line of code to see how fast it runs (it runs it many times to get an average).
3. **Tab Completion**: Start typing a variable and hit `Tab` to see all available methods.
4. **Shift + Tab**: Hit this while your cursor is inside a function's parentheses to see its documentation!
5. **! (Bang operator)**: Use `!pip install ...` or `!ls` to interact with your computer's terminal directly from a cell.
