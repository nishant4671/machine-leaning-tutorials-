# MLflow Overview 📈

Welcome to the lab manager's office! 

### What is MLflow?
MLflow is an open-source platform to manage the ML lifecycle. It helps you track experiments, package code into reproducible runs, and share/deploy models.

### Why use it?
Have you ever forgotten which parameters gave you that 95% accuracy last week? MLflow remembers everything for you!

### Core Concepts
1. **Tracking**: Logging parameters, code versions, metrics, and output files.
2. **Projects**: A standard format for packaging reusable data science code.
3. **Models**: A convention for packaging machine learning models.

### Installation
```bash
pip install mlflow
```

### Alternatives
- **Weights & Biases (W&B)**: Very popular cloud-based tracking.
- **DVC**: Great for versioning large data files.
