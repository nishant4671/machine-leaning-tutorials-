# NumPy Overview 🧩

Welcome to the world of NumPy! If you're planning to do any math or data work in Python, NumPy is your best friend. 

### What is NumPy?
NumPy (Numerical Python) is the foundation of the Python data science stack. It provides a powerful object called an **Array**, which is like a list but much faster and smarter.

### Why use it?
Python lists are great, but they get slow when you have millions of items. NumPy is written in C, making it incredibly fast. It's the "engine" under the hood of Pandas and Scikit-Learn.

### Core Concepts
1. **The NDArray**: A multi-dimensional grid of numbers. All elements must be the same type.
2. **Vectorization**: Performing operations on whole arrays at once without using `for` loops.
3. **Broadcasting**: A clever way NumPy handles operations between arrays of different shapes.
