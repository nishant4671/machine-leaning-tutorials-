# AI Teaching Instructions — Module 4: Recommender Systems & LLM Agents

## 🧑‍🏫 Role
> "You are a specialist in recommender systems and AI agent development. Your student is building an e‑commerce product recommender with a local LLM agent for internal tooling. You emphasise scalable serving, A/B testing, and safe AI agent design."

## 📚 Prompts

- "Explain collaborative filtering with a movie rating example. What’s the difference between user‑based and item‑based?"
- "What is matrix factorisation? How does it learn latent factors? Use a story of movie genres."
- "Teach me FAISS. Why is it fast? Give me a minimal code example to index and search embeddings."
- "Design an A/B testing system for my recommender API using feature flags. How do I route 10% of traffic to a new model?"
- "I want to build an LLM agent that can query a database and fetch MLflow metrics. Use LangChain and a local Llama3 via Ollama. Provide a skeleton with tool definitions."
- "What are prompt injection attacks? How do I protect my internal agent? Give me a safe system prompt and validation rules."
- "My agent suggests product deletions. Help me restrict its actions to read‑only. Show me how to filter tool calls."
- "Docker Compose: help me set up a multi‑service environment (API, Postgres, Redis, Ollama). Review my docker-compose.yml for networking and health checks."
- "Conduct a mock system design interview: ‘Design a real‑time recommendation system for 10 million users.’ Give me feedback on my answer."
