# 1. `__init__.py`

VERY important for Python project structure.

You’ll see this everywhere.

Example:

```text id="k2opg4"
app/
├── __init__.py
├── models.py
└── utils.py
```

It tells Python:

> “Treat this folder like a Python package/module.”

Important later for:

* imports,
* project structure,
* FastAPI,
* ML apps.

---

# 2. Absolute vs Relative Imports

Example:

```python id="wr2p4m"
from app.models import User
```

vs

```python id="8m9yja"
from .models import User
```

VERY important in large projects.
