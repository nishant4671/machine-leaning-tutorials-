# Linear Regression Concepts

## Intuition

Linear Regression is the simplest machine learning algorithm.

Its goal is to learn a relationship between an input and an output.

Example:

Hours Studied ? Exam Score

The model attempts to find a line that best fits the data.

y = mx + b

Where:

- x = input feature
- y = prediction
- m = slope
- b = intercept

---

## Why We Need It

Suppose we know:

1 hour ? 3 marks
2 hours ? 5 marks
3 hours ? 7 marks

We want to predict:

4 hours ? ?

Linear Regression learns the pattern and predicts unseen values.

---

## Core Idea

Machine Learning is:

Input
? Prediction
? Error
? Improvement

Linear Regression is the simplest example of this process.

---

## Assumptions

Linear Regression assumes:

- Relationship is approximately linear
- Data contains a learnable pattern
- Small changes in x produce proportional changes in y

---

## Common Mistakes

Mistake:

Thinking ML memorizes data.

Reality:

The model learns parameters.

For Linear Regression:

- slope
- intercept

---

## Industry Usage

Linear Regression is still used for:

- Forecasting
- Pricing models
- Trend analysis
- Baseline models

---

## Future Connections

Linear Regression leads directly to:

- Logistic Regression
- Neural Networks
- Deep Learning

Neural networks are essentially large collections of learned parameters.

---

## Revision Summary

Linear Regression learns:

y = mx + b

by minimizing prediction error.
