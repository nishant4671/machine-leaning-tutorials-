# Unit 0.4 - ML in Code

## Chapter Objective

This chapter bridges the gap between machine learning theory and machine learning implementation.

The goal is to understand:

- Linear Regression
- Prediction and Error
- Mean Squared Error (MSE)
- Gradient Descent
- Learning Rate
- Iterations
- Loss Curves
- Normal Equation
- sklearn LinearRegression
- StandardScaler
- Pipelines
- Model Persistence with joblib

The student should understand both:

1. How ML works internally
2. How ML is used in industry

---

## Learning Outcomes

After completing this chapter, you should be able to:

- Explain linear regression intuitively
- Compute MSE manually
- Explain why gradient descent works
- Implement simple linear regression with NumPy
- Use sklearn LinearRegression
- Use StandardScaler
- Build sklearn Pipelines
- Save and load models using joblib
- Distinguish training from inference
