# ML In Code - One Page Revision

## Linear Regression

Purpose:

Learn relationship between input and output.

Equation:

y = mx + b

m = slope

b = intercept

---

## Prediction

Prediction:

y_hat

Goal:

Make predictions close to actual values.

---

## Error

Error = Actual - Predicted

---

## MSE

Mean Squared Error

MSE = Mean((Actual - Predicted)^2)

Lower is better.

---

## Gradient Descent

Purpose:

Reduce loss.

Loop:

Predict

?

Error

?

Loss

?

Update Parameters

?

Repeat

---

## Learning Rate

Controls update size.

Too Small:

Slow learning.

Too Large:

Unstable learning.

---

## Training

Model learns parameters.

---

## Inference

Model makes predictions.

No learning occurs.

---

## StandardScaler

Puts features on similar scales.

---

## Pipeline

Automates preprocessing + model steps.

---

## joblib

Save model:

joblib.dump()

Load model:

joblib.load()

---

## Most Important Mental Model

Machine Learning = Repeated Error Correction
