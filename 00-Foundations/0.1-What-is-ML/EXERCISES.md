# Unit 0.1: What is Machine Learning?

## 🎯 Learning Objectives
Understand supervised/unsupervised/reinforcement, the ML workflow, overfitting, underfitting, bias-variance, and train/val/test split.

---

## 🧪 Exercises
1. **Write a one‑paragraph explanation** of each type of learning (supervised, unsupervised, reinforcement) in your learning-log.md. Use a real‑world analogy for each.

2. **Draw the ML workflow** on paper or using a digital tool. Label: Raw Data → Features → Model Training → Evaluation → Deployment → Monitoring. Take a photo/screenshot and save it in this folder.

3. **Overfitting demo**: Watch the first 5 minutes of [this StatQuest video](https://www.youtube.com/watch?v=EuBBz3bI-aA). Then write in your own words: What is overfitting? How does a train/validation/test split help detect it?

4. **Bias-Variance tradeoff**: Explain it to a rubber duck. Record a 2‑minute voice memo or write a short paragraph. No one will read it but you.

---

## ✅ Completion Checklist
*You have completed this unit when…*
I can give a real‑world example for each ML type without looking at notes.
